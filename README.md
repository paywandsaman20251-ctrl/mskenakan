# Msken Kingdom Website

A fun friend-group website where:

- Shex and Baag are leaders
- Msken and the Mskenms are servants
- Every member has an image (recommended ImgBB direct link)

## Run locally

1. Install dependencies:
   `npm install`
2. Create `.env` file from `.env.example`
3. Add your MongoDB connection string in `MONGODB_URI`
4. Start:
   `npm start`
5. Open:
   `http://localhost:3000`

## Add members

Use the form on the home page to add:

- Name
- Role (`leader` or `servant`)
- Image URL (ImgBB link)
